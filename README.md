# Nikkblend - Luxury Hair & Skin Services

A premium, mobile-friendly salon website built with React, Next.js, TailwindCSS, and Framer Motion.

## Features

- **Responsive Design**: Mobile-first approach for all devices
- **Luxury Minimal Aesthetic**: Black, Gold, and White color palette
- **Smooth Animations**: Framer Motion for engaging interactions
- **Multiple Pages**: Home, Services, Transformations, Gallery, About, Book, Contact
- **WhatsApp Integration**: Direct booking via WhatsApp
- **Floating CTA Button**: Always-accessible booking button
- **Professional Styling**: Playfair Display and Poppins typography

## Pages

- **Home**: Hero section with services overview and testimonials
- **Services**: Detailed service listings for male and female clients
- **Transformations**: Gallery of before/after transformations
- **Gallery**: Instagram-style image gallery
- **About**: Professional introduction and why choose us
- **Book**: Appointment booking form and WhatsApp integration
- **Contact**: Contact information, social links, and service area map

## Tech Stack

- **Frontend**: React 19 + TypeScript
- **Styling**: TailwindCSS 4
- **Animations**: Framer Motion
- **Routing**: Wouter
- **Icons**: Lucide React
- **Build Tool**: Vite

## Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview
```

## Development

The project follows a modular component structure:

```
client/
├── src/
│   ├── pages/        # Page components
│   ├── components/   # Reusable components
│   ├── App.tsx       # Main routing
│   └── index.css     # Global styles
├── public/           # Static assets
└── index.html        # HTML entry point
```
